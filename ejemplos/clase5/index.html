¿Cómo evitar que el contenedor colapse?

Dos opciones:

.container {
  overflow: hidden;
}

O con el hack de clearfix:

.clearfix:after {
   content: " ";
   visibility: hidden;
   display: block;
   height: 0;
   clear: both;
}

O esta opción más corta para IE ≥ 8:

.clearfix:after {
  content: "";
  display: table;
  clear: both;
}
